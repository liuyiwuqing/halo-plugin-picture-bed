# 图床插件用户使用指南

本文面向 Halo 站点管理员，说明如何安装、配置和使用图床插件。

## 使用前确认

请先确认：

- Halo 版本不低于 `2.25.0`。
- 插件已安装并启用。
- 当前账号拥有 `plugin:picturebed:manage` 权限。
- 已准备好对应图床平台的 API 地址和认证信息。

如果你不是超级管理员，看不到菜单时先别怀疑人生，大概率就是角色权限没给。

## 快速配置

1. 进入 Halo 控制台。
2. 打开 `插件`，找到 `图床插件` 并启用。
3. 进入插件设置页，打开 `基本设置`。
4. 在 `图床接口` 中新增一条配置。
5. 选择图床类型，填写图床名称、API 地址和认证信息。
6. 打开 `是否启用`。
7. 保存设置并刷新控制台。
8. 进入 `工具 -> 图床管理` 开始使用。

一个站点可以配置多个图床实例，也可以配置同一种图床的多个账号。建议给实例起清楚的名字，例如“文章内容图床”“首页封面图床”“临时素材图床”。

## 通用配置项

| 配置项       | 是否必填 | 说明                                                                |
|-----------|------|-------------------------------------------------------------------|
| 图床 ID     | 自动生成 | 用于区分同类型下的多个图床实例                                                   |
| 图床类型      | 必填   | 可选 `兰空图床`、`SM.MS图床`、`ImgTP图床`、`123盘图床`                            |
| 图床名称      | 必填   | 展示在图床管理页和附件选择器中                                                   |
| 是否启用      | 必填   | 只有启用的实例才会出现在管理页和附件选择器中                                            |
| API 地址    | 必填   | 第三方图床 API 基础地址，通常需要保留末尾 `/`                                       |
| API Token | 必填   | 第三方图床访问令牌。123 盘当前主要使用 `client_id` 和 `client_secret`，但设置表单仍要求保留该字段 |

## 各图床配置

### 兰空图床 Lsky Pro

推荐 API 地址格式：

```text
https://your-lsky.example/api/v1/
```

需要填写：

- `API 地址`
- `API Token`
- `储存策略 ID`，可选

能力说明：

- 支持相册列表。
- 支持按相册浏览图片。
- 支持关键字搜索。
- 支持上传和删除。
- 如果填写储存策略 ID，上传时会把图片提交到对应储存策略。

### SM.MS

推荐 API 地址格式：

```text
https://sm.ms/api/v2/
```

需要填写：

- `API 地址`
- `API Token`

能力说明：

- 使用 SM.MS 上传历史作为图片列表。
- 支持分页浏览、上传和删除。
- 不提供相册能力。
- 当前管理页不提供关键字搜索入口。

### ImgTP

需要填写：

- `API 地址`
- `API Token`

能力说明：

- 支持图片列表、上传和删除。
- 不提供相册能力。
- 当前管理页不提供关键字搜索入口。

如果 ImgTP 接口地址不是标准基础地址，列表或删除可能会失败。先确认你填写的 `API 地址` 与当前 ImgTP 服务的接口路径拼接规则一致。

### 123 盘图床

需要填写：

- `API 地址`
- `API Token`
- `client_id`
- `client_secret`

能力说明：

- 图片列表和删除使用 123 盘 OpenAPI。
- 文件夹会以分组方式展示。
- 列表通过 `查看更多` 加载后续批次。
- 不支持当前插件里的常规关键字搜索和普通分页。
- 当前 Console 管理页没有开放上传按钮。
- 插件会按 `client_id` 缓存访问令牌，多个 123 盘配置不会串用同一个 token。

## 图床管理页

进入路径：

```text
工具 -> 图床管理
```

页面顶部可以切换已启用的图床实例。切换后，页面会加载对应图床的图片列表和可用分组。

### 浏览图片

管理页支持两种展示模式：

- `网格`：卡片尺寸稳定，适合快速扫描。
- `瀑布流`：保留图片比例，适合查看不同尺寸的素材。

点击图片可以选中或取消选中。点击预览按钮可以打开详情弹窗，查看文件名、媒体类型、大小和链接。

### 上传图片

支持上传入口的图床：

- 兰空图床 Lsky Pro
- SM.MS
- ImgTP

当前上传组件允许的格式：

```text
.jpg, .jpeg, .png, .gif, .svg, .bmp, .webp
```

上传失败时，优先检查：

- API 地址是否正确，末尾 `/` 是否符合图床平台要求。
- Token 或认证信息是否有效。
- 图床服务端是否能从 Halo 服务访问。
- 图片大小是否超过第三方平台限制。

### 选择图片

你可以：

- 单选图片。
- 多选图片。
- 全选当前列表中可选图片。
- 在详情弹窗中切换上一张或下一张。

当插件用于 Halo 附件选择器时，选择结果会以附件形式返回给调用方，包含展示名称、媒体类型、文件大小和外部链接。

### 删除图片

删除步骤：

1. 勾选一张或多张图片。
2. 点击 `删除`。
3. 在确认弹窗中确认。
4. 等待接口返回结果。

注意：

- 删除会调用第三方图床接口，不是只从 Halo 页面移除。
- 删除成功后通常不可恢复。
- 批量删除会逐张调用接口，部分失败时页面会提示失败数量。

## 附件选择器联动

插件会向 Halo 附件选择器注册入口。满足以下条件时才会显示：

- 当前账号拥有 `plugin:picturebed:manage` 权限。
- 图床实例已启用。
- 图床类型在当前插件中有对应 Provider。

常见使用场景：

- 在文章编辑器中插入图床图片。
- 在主题设置中选择外部图片。
- 在其他插件表单的附件字段中选择图床资源。

## 权限配置

如果普通用户需要使用图床，请在 Halo 角色权限中分配“图床管理操作”。

该权限会影响：

- 是否能看到 `工具 -> 图床管理` 菜单。
- 是否能调用插件图床接口。
- 附件选择器中是否展示图床入口。

权限没配好时，插件不是坏了，是你让它干活又不给钥匙，这事儿它也挺冤。

## 常见问题

### 保存配置后看不到图床实例

按顺序检查：

- `是否启用` 是否打开。
- 插件设置是否保存成功。
- 当前账号是否拥有图床管理权限。
- 是否刷新过 Halo 控制台页面。
- 图床类型是否是当前插件支持的四种类型之一。

### 看得到菜单，但图片列表为空

可能原因：

- 第三方图床账号本身没有图片。
- API 地址或 Token 不正确。
- 第三方平台返回超时或拒绝访问。
- 当前图床类型不支持你期望的相册、搜索或分页能力。
- 123 盘需要点击 `查看更多` 加载后续批次。

### 上传失败

优先检查：

- 图片格式是否在允许列表中。
- 图床平台是否限制单文件大小。
- API 地址是否能从 Halo 所在服务器访问。
- Token、`client_id`、`client_secret` 是否正确。
- 兰空图床的储存策略 ID 是否存在且可用。

### 附件选择器里没有图床入口

检查：

- 图床实例是否启用。
- 当前账号是否拥有 `plugin:picturebed:manage` 权限。
- 图床类型是否有对应 Provider。
- 页面是否已经刷新。

### 为什么有些图床没有相册

因为第三方平台能力不同：

- 兰空图床有相册。
- 123 盘以文件夹方式展示。
- SM.MS 和 ImgTP 当前按图片列表展示，没有相册。

### 为什么 123 盘没有搜索和分页

当前实现使用 123 盘 OpenAPI 的文件列表能力，通过 `lastFileId` 批次加载。它和常规页码分页不是一回事，所以页面使用
`查看更多`。

### 删除后图片还短暂显示

页面会在删除后刷新列表，但第三方平台或浏览器缓存可能让图片短时间仍可访问。确认接口返回成功后，以第三方平台最终状态为准。

## 使用建议

- 生产环境不要使用临时、测试或会频繁变动的 API 地址。
- 给每个图床实例使用清晰名称，避免编辑内容时选错。
- 只把图床管理权限分配给可信角色。
- 删除前确认图片是否仍被文章、页面、主题或插件引用。
- 第三方图床 Token 建议定期轮换，离职、泄露或权限变更后要立即更换。
