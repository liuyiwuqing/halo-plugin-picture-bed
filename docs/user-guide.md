# 图床插件使用说明

## 1. 插件简介

图床插件用于把 Halo 后台的图片上传、选择和删除操作接入第三方图床服务。安装完成后，你可以：

- 在 Halo 后台统一管理图床中的图片
- 在附件选择器里直接从图床选图
- 对不同业务场景配置多个图床接口

当前版本支持以下图床类型：

- 兰空图床（Lsky Pro）
- SM.MS
- ImgTP
- 123 盘图床

## 2. 使用前准备

请先确认以下条件：

- Halo 版本不低于 `2.22.0`
- 插件已经在 Halo 中安装并启用
- 当前账号拥有“图床管理”权限
- 你已经准备好对应图床的 API 地址和认证信息

如果你不是站点管理员，请让管理员为你的角色分配“图床管理”相关权限。

## 3. 安装与启用

1. 在 Halo 控制台进入“插件”。
2. 安装图床插件并启用。
3. 打开插件设置页，进入“基本设置”。
4. 在“图床接口”里新增至少一条配置并保存。

保存成功后，后台左侧“工具”分组下会出现“图床管理”菜单；附件选择器里也会显示已启用的图床。

## 4. 配置说明

每条图床配置都对应一个独立的图床入口。一个站点可以同时配置多个不同类型，或者同一类型的多个账号。

### 通用字段

| 字段        | 说明                       |
|-----------|--------------------------|
| 图床 ID     | 自动生成，用于区分同类型下的不同配置       |
| 图床类型      | 选择当前接入的是哪一种图床            |
| 图床名称      | 后台和附件选择器中展示的名称           |
| 是否启用      | 只有启用后的图床才会出现在管理页面和附件选择器中 |
| API 地址    | 图床 API 基础地址，通常要求以 `/` 结尾 |
| API Token | 图床提供的访问令牌                |

### 兰空图床（Lsky Pro）

需要填写：

- API 地址：例如 `https://your-lsky.example/api/v1/`
- API Token：兰空后台生成的 Token
- 储存策略 ID：可选；如果你在兰空中配置了多种存储策略，可以指定上传走哪一个

说明：

- 兰空支持相册列表，因此在管理页里可以切换相册查看图片

### SM.MS

需要填写：

- API 地址：例如 `https://sm.ms/api/v2/`
- API Token：SM.MS 提供的 Token

说明：

- SM.MS 以历史上传记录为主，不提供和兰空完全一致的相册能力

### ImgTP

需要填写：

- API 地址：填写 ImgTP API 基础地址，并保留末尾 `/`
- API Token：ImgTP 提供的 Token

说明：

- ImgTP 管理页支持查看、上传、选择和删除

### 123 盘图床

需要填写：

- API 地址：填写你的 123 盘上传服务基础地址，插件会在其后追加 `upload`
- API Token：当前实现不会用它做 OpenAPI 鉴权，但字段仍需保留
- client_id：123 云盘 OpenAPI 的 `client_id`
- client_secret：123 云盘 OpenAPI 的 `client_secret`

说明：

- 图片列表和删除走 123 OpenAPI
- 上传走你配置的上传服务地址
- 123 盘接口不支持当前插件里的常规关键词搜索和普通分页，因此管理页使用“查看更多”的方式继续加载
- 如果你配置了多个 123 盘接口，现在可以分别缓存各自的访问令牌，不会再串用

## 5. 日常使用

### 进入管理页

进入 Halo 控制台后，打开：

- `工具 -> 图床管理`

页面顶部可以切换当前使用的图床实例。

### 上传图片

1. 选择一个图床实例。
2. 点击“上传”。
3. 选择本地图片文件。
4. 上传完成后刷新列表即可查看。

当前上传组件支持的格式：

- `.jpg`
- `.jpeg`
- `.png`
- `.gif`
- `.svg`
- `.bmp`
- `.webp`

### 选择图片

在图床管理页或附件选择器中，你可以：

- 单选或多选图片
- 全选当前页/当前批次图片
- 打开详情查看图片信息

### 删除图片

1. 勾选一张或多张图片。
2. 点击“删除”。
3. 在确认弹窗中确认操作。

现在删除操作会等待接口返回结果后再刷新列表，并给出成功/失败提示。

## 6. 与 Halo 附件选择器的联动

当图床被启用且当前账号具备权限时，Halo 附件选择器会自动出现对应的图床入口。

常见使用方式：

- 在文章编辑器中直接插入图床图片
- 在主题设置、插件表单等附件字段中选择图床资源

附件选择器里只会显示“已启用”的图床配置。

## 7. 常见问题

### 1）保存了图床配置，但页面里看不到

请依次检查：

- 是否将“是否启用”设置为“是”
- 当前账号是否具备“图床管理”权限
- 是否已经保存插件设置
- 是否刷新了当前 Halo 后台页面

### 2）上传时报错

建议优先检查：

- API 地址是否正确，是否需要以 `/` 结尾
- Token、`client_id`、`client_secret` 是否填写正确
- 对应图床服务端是否允许当前来源访问
- 123 盘上传地址是否可用

### 3）切换到 123 盘后以前看不到内容

当前版本已经补齐 123 盘在管理页中的组件支持；如果仍然看不到，请检查：

- 图床类型是否确实选择了“123盘图床”
- 配置是否启用
- `client_id` 和 `client_secret` 是否有效

### 4）为什么有些图床没有相册切换

不同平台 API 能力不同。当前插件里：

- 兰空图床支持相册列表
- 123 盘以文件夹和“查看更多”方式展示
- SM.MS、ImgTP 以图片列表管理为主

## 8. 使用建议

- 给不同业务场景配置不同图床名称，方便在附件选择器中区分
- 如果同一类型配置多个实例，建议名称中标明用途，例如“文章封面图床”“评论图片图床”
- 生产环境下优先使用稳定、长期可用的 API 地址
- 修改图床配置后，建议回到“图床管理”页手动刷新一次确认生效
